
# JavaScript Mini Projects – Chai Aur Code

This repository contains beginner-friendly JavaScript projects built while following the Chai Aur Code tutorials. Each project focuses on core JavaScript concepts like DOM manipulation, event handling, basic logic building, and UI interaction.

--------------------------------------------------

## Projects Included

• Project 1 – Color Changer  
  - Changes background color dynamically on button click  
  - Helps understand DOM selection and event handling  
  - Instant UI update using JavaScript  

• Project 2 – BMI Generator  
  - Calculates BMI using height and weight inputs  
  - Shows health category based on result  
  - Demonstrates form handling and input validation  

• Project 3 – Digital Clock  
  - Displays real-time current time  
  - Updates automatically every second  
  - Built using Date object and setInterval()  

• Project 4 – Guess The Number  
  - Random number guessing game  
  - Gives hints (higher/lower) after each guess  
  - Limited attempts with game result
